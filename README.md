# Custom Object Display System
This is a custom object display system for Roblox written in Luau.


### Usage
An example local script is given on how to use this module. The example local script is child of a GUI and has a sibling folder called "Objectviewports" (Automatically created non-existant.)
Place the GUI with the example local script, inside of StarterGui.<br>
**Make sure Reset on spawn is off!**
